= About Administrator Tasks

include::_attributes.adoc[]
endif::[]

Exchange administrators can perform the following tasks:

* xref:to-set-permissions.adoc[Set Exchange role permissions] - Determine who in your organization should be an Exchange Administrator, Contributor, or Viewer. You can also provide role access for individual assets.
* xref:to-save-searches.adoc[To Save Searches] - Save and maintain searches for assets within a business group.
* xref:to-share-an-asset.adoc[To Share an Asset] - Grant access to view an asset to users in your organization or to a chosen external organization. You can also use Share to publish an API asset to a Public Portal.
* xref:to-enable-the-request-access-button.adoc[Use API Manager to add the Request Access button] - This enables you to specify which API assets can be shared with users.
* xref:to-deprecate-asset.adoc[To Deprecate an Asset] - Remove access to a version within an asset but keep the asset available. Users can access other versions within the asset.
* xref:migrate.adoc[To Migrate Exchange 1 Assets to Exchange 2] - How to handle assets moved from Exchange 1.
